# Beecrowd Challenge 1007 - Difference

## 📋 Problem Description
- **Problem ID:** 1007
- **Title:** Difference  
- **Difficulty:** Beginner
- **Link:** [View Problem](https://judge.beecrowd.com/en/problems/view/1007)

### Problem Statement
Read four integer values (A, B, C, D) and calculate the difference: (A × B - C × D), then print the result in the format: `DIFERENCA = {difference}`

## 💡 Solution Approach

### Strategy
1. Read four integer inputs (A, B, C, D)
2. Calculate the difference: (A × B) - (C × D)
3. Print the result in the required format

## 🔍 Code Explanation

| Component | Purpose |
|-----------|---------|
| `int(input())` | Reads input and converts to integer |
| `(A * B - C * D)` | Calculates the difference between products |
| `print(f"DIFERENCA = {DIFERENCA}")` | Outputs result in required format |
