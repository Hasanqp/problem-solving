## 📋 Problem Description
- **Problem ID:** 1004
- **Title:** Simple Product  
- **Difficulty:** Beginner
- **Link:** [View Problem](https://judge.beecrowd.com/en/problems/view/1004)

### Problem Statement
Read two integer values, calculate their product, and print the result in the format: `PROD = {product}`

## 💡 Solution Approach

### Strategy
1. Read two integer inputs from the user
2. Calculate their product
3. Print the result in the required format

## 🔍 Code Explanation

| Component | Purpose |
|-----------|---------|
| `Convert.ToInt32(Console.ReadLine())` | Reads input and converts to integer |
| `A * B` | Performs multiplication operation |
| `Console.WriteLine("PROD = " + PROD)` | Outputs result in required format |
