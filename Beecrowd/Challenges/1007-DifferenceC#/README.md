## 📋 Problem Description
- **Problem ID:** 1007
- **Title:** Difference  
- **Difficulty:** Beginner

### Problem Statement
Read four integer values (A, B, C, D) and calculate the difference: (A × B - C × D), then print the result in the format: `DIFERENCA = {difference}`

## 💡 Solution Approach

### Strategy
1. Read four integer inputs (A, B, C, D)
2. Calculate the difference: (A × B) - (C × D)
3. Print the result in the required format

## 🔍 Code Explanation

| Component | Purpose |
|-----------|---------|
| `Convert.ToInt32(Console.ReadLine())` | Reads input and converts to integer |
| `(A * B - C * D)` | Calculates the difference |
| `Console.WriteLine("DIFERENCA = " + DIFERENCA)` | Outputs result in required format |
