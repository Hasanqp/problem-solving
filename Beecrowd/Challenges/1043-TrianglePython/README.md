## 🔍 Code Explanation

| Component | Purpose |
|-----------|---------|
| `map(float, input().split())` | Reads and converts three floats |
| `A + B > C and A + C > B and B + C > A` | Triangle inequality check |
| `A + B + C` | Perimeter calculation |
| `((A + B) * C) / 2` | Trapezium area calculation |
| `{perimeter:.1f}` or `{area:.1f}` | Formats output to 1 decimal place |

## 📐 Triangle Inequality Theorem

A valid triangle must satisfy:
- A + B > C
- A + C > B
- B + C > A

If all conditions are true → Triangle
Otherwise → Trapezium
