# Beecrowd Challenge 1059 - Even Numbers

## 📋 Problem Description
- **Problem ID:** 1059
- **Title:** Even Numbers  
- **Difficulty:** Beginner
- **Link:** [View Problem](https://judge.beecrowd.com/en/problems/view/1059)

### Problem Statement
Print all even numbers between 1 and 100, inclusive.

## 💡 Solution Approach

### Strategy
1. Start from 2 (first even number)
2. Increment by 2 each time
3. Continue until 100
4. Print each number on a new line

## 🔍 Code Explanation

| Component | Purpose |
|-----------|---------|
| `for (int i = 2; i <= 100; i += 2)` | Loop from 2 to 100, step 2 |
| `Console.WriteLine(i)` | Prints each even number |
